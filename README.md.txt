Καλησπέρα! Ονομάζομαι Κωνσταντίνος Καλογιάννης και αυτή είναι η ατομική μου εργασία στην ΑΔΙΣΕ.
Το παρόν project υλοποιεί το παιχνίδι καρτών 'Ξερή' ως Web API χωρίς GUI με χρήση PHP και SQL.
Οι παίχτες χρησιμοποιούν το παιχνίδι μέσω CLI με εντολές κυρίως curl.
Η κατάσταση του παιχνιδιού αποθηκεύεται στην εξ ολοκλήρου SQL και κάθε παίχτης μπορεί να 
συμμετέχει μόνο μέσω HTTP endpoints που επιστέφουν JSON.

---

Αναλυτικά τα χαρακτηριστικά της εργασίας.
-Υλοποίηση του παιχνιδιού Ξερής human vs human.
-Χωρίς GUI.
-Χρήση CLI με curl.
-Χρήση:
  -PHP.
  -MySQL.
  -JSON API.
-Authentication χωρίς  password(γίνεται από αυτόματα από το auth.php).
-Έλεγχος σειράς παίχτης που παίζει.
-Έλεγχος κανόνων Ξερής(πιασίματα,ρηγάδες,δεκάρια,ξερή).
-Deadlock detection(τέλος τράπουλας και άδειο χέρι).
-Αναμονή κίνησης αντίπαλου.


---

Κανόνες Ξερής (Συνοπτικά)

- Μοιράζονται **4 κάρτες στον κάθε παίκτη** και **4 στο τραπέζι** (η κάτω δε γυρίζεται).
- Ο παίκτης παίζει μία κάρτα κάθε σειρά.
- Πιάσιμο:
  - Ίδια αξία με την τελευταία κάρτα στο τραπέζι → παίρνει όλο το τραπέζι
  - Βαλές (J) → παίρνει όλο το τραπέζι
- Ξερή:
  - Πιάσιμο όταν υπάρχει **μόνο 1 κάρτα** στο τραπέζι
- Πόντοι:
  - 10άρια = 1 πόντος
  - Ρηγάδες = 1 πόντος
  - Ξερές = 1 πόντος η καθεμία
  - Πλειοψηφία χαρτιών = 3 πόντοι
- Τέλος παιχνιδιού:  
  Όταν τελειώσει η τράπουλα και οι δύο παίκτες δεν έχουν άλλο φύλλο.


---
Περιεχόμενο της MySQL (schema.sql)

Το schema περιλαμβάνει τους πίνακες:

- games— βασικές πληροφορίες παιχνιδιού
- cards - όλες οι κάρτες της τράπουλας και το value τους
- players — players με tokens
- moves — όλες οι κινήσεις (ιστορικό)
- deck— η τράπουλα όπως μοιράστηκε
- state — πλήρης κατάσταση board + χεριών

Η βάση μπορεί να αναδημιουργήσει το παιχνίδι ανά πάσα στιγμή.

---

Commands curl:
curl -X POST http://localhost/xeri/public/game_create.php
Δημιουργία Παιχνιδιού. Ο πρώτος παίκτης περιμένει αντίπαλο.

curl -X POST http://localhost/xeri/public/game_join.php?game_id=1
Ο δεύτερος παίκτης εκανε join.

curl "http://localhost/xeri/public/state.php?game_id=1&player_id=100&token=abc123"
Ελεγχος κατάστασης του παιχνιδιού.(Εμφανίζει χέρι παίκτη,τραπέζι,σειρά,πόσες κάρτες απομένουν,
πόντους,αν υπάρχει νικητής).

curl -X POST \ -d "game_id=1&player_id=100&token=abc123&card=7H" \ http://localhost/xeri/public/play.php
Παίξιμο κάρτας.

curl "http://localhost/xeri/public/poll.php?game_id=1&player_id=101&token=xyz555"
Αναμονή σειριάς.(ο client περιμείνει κοντά στα 30 δευτερόλεπτα).

curl "http://localhost/xeri/public/state.php?game_id=1&player_id=100&token=abc123"
Όταν τελειώσει η τράπουλα και δεν υπάρχουν κινήσεις.

curl -X POST \ -d "game_id=1&player_id=101&token=xyz555" \ http://localhost/xeri/public/forfeit.php
Παραίτηση παιχνιδιού.

curl POST play.php 
Ο παίκτης με σειρά.

curl poll.php  και curl state.php
Ο άλλος παίκτης κάνει.

-Συνιστώμενη Ροή Παιχνιδιού

1)Παίκτης 1 → game_create
2)Παίκτης 2 → game_join-
3)Και οι δύο → state
4)Παίκτης που έχει σειρά → play
5)Άλλος παίκτης → poll
6)Επανάληψη μέχρι:
7)Τέλος τράπουλας
8)Deadlock
9)Παραίτηση
10)Τελικό state → εμφάνιση νικητή.
